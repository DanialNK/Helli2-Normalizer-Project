**یک کلمه ای که خودم از خودم تولید کردم به معنی اینه که بیای و یک تابع تعریف کنید که توش از چندین توابع استفاده شده که صرفا برای خلاصه کردن کار تعریف میشود
مثال:**
~~~python
def tabeh1(w1):
	w1=w1+'ريال'
	
def tabeh2(w2):
	w2=w2+'تومان'
	
def magma_al_tavabeh(w1,w2):
	return (tabeh1(w1) + tabeh2(w2))
~~~
